# Portfolio (wOs)

Personal portfolio site with admin panel.

- **Documentation:** [docs/README.md](docs/README.md)
- **Main site:** `index.html`
- **Admin:** `admin.html`

## Quick start

```bash
npm install
npm run dev
```

- **`npm run dev`** — [live-server](https://www.npmjs.com/package/live-server) on port **8000** with **live reload** (browser refreshes when you save HTML, CSS, or JS).
- **`npm run dev:static`** — plain `http-server` (no auto-refresh), same as before.

Open `http://127.0.0.1:8000` (or the URL printed in the terminal).
